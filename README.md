# Landing Page Template

Website sederhana dengan routing ke iframe template HTML.

## Fitur
- `/` - Halaman utama
- `/1` - Preview template Durbarinn dalam iframe
- `/2` - Preview template Eventiz dalam iframe

## Cara Deploy
1. Upload semua file ke GitHub repository
2. Aktifkan GitHub Pages di repository settings
3. Pilih source dari main branch

## Local Development
Buka `index.html` di browser atau gunakan live server.
